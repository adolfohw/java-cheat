The Basics

About

Java is a garbage-collected, object-oriented, C-like™ language that compiles into bytecode, which is then run on the JVM. This means that it is all about classes, and that Java apps are very portable. Allegedly.

The JDK can be downloaded from Oracle's website.

  • javac HelloWorld
  • java HelloWorld

Tooling

üêò Gradle

Gradle is a common build tool in the Java world, and can be downloaded from its website. To start a new Java project:

  • mkdir java-IS-cool
  • cd java-IS-cool
  • gradle init

At this point, it's recommended to use the gradlew scripts generated in the project's root. These are some useful tasks:

  • ./gradlew build   # builds your app and runs tests
  • ./gradlew jar     # makes a jar archive of the project
  • ./gradlew run     # runs your app
  • ./gradlew test    # runs unit tests
  • ./gradlew tasks   # shows all tasks!
